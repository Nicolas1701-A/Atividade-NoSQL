# Projeto NoSQL com MongoDB Atlas

Este projeto demonstra um sistema simples em Python utilizando MongoDB Atlas com PyMongo, incluindo operações de CRUD e simulação de vendas com atualização de estoque.

---

## 🚀 Tecnologias utilizadas

- Python 3.x
- MongoDB Atlas
- PyMongo

---

## 📦 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/projeto_nosql.git
cd projeto_nosql

2. Criar ambiente virtual (venv)
python -m venv venv

3. Ativar o ambiente virtual
Windows:
.\venv\Scripts\activate

4. Instalar dependências
pip install pymongo

Ou, se existir o arquivo:

pip install -r requirements.txt

5. Executar o projeto
python app.py

requirements.txt
pymongo
